# Cómo correr

1. Primero descarga el repositorio
2. Descarga el dataset de [aquí](https://drive.google.com/drive/folders/1Mo4yF2PrBKXgiPFyHAsp5hB6bkkPRxbk)
3. Descomprime el dataset en la raíz del repositorio
4. Instala las dependencias con `pip install -r requirements.txt`
5. Corre el jupyter notebook main.ipynb
